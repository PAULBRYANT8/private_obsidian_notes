  query 2 [✓  ✓   ✓  ✗  ✗  ✗  ✗  ✗]

  query 5 [✓  ✓   ✓  ✓  ✓  ✓  ✗  ✗]                                                                                                                                                                                  
  query 6 [✓  ✓   ✓  ✓  ✓  ✓  ✓  ✗]

  Band 模式（滑动窗口 attention，window_size=3）                                                                                                                                                                    
                                                                                                                                                                                                                    
  Band 模式限制每个 query 只能看到离自己最近的 window_size 个 token：                                                                                                                                               
                  
           kv 位置                                                                                                                                                                                                  
           0  1  2  3  4  5  6  7             
  query 0 [✓  ✗  ✗  ✗  ✗  ✗  ✗  ✗]                                                                                                                                                                                  
  query 1 [✓  ✓  ✗  ✗  ✗  ✗  ✗  ✗]
  query 2 [✓  ✓  ✓  ✗  ✗  ✗  ✗  ✗]                                                                                                                                                                                  
  query 3 [✗  ✓  ✓  ✓  ✗  ✗  ✗  ✗]  ← 位置 0 超出窗口，被丢弃
  query 4 [✗  ✗  ✓  ✓  ✓  ✗  ✗  ✗]                                                                                                                                                                                  
  query 5 [✗  ✗  ✗  ✓  ✓  ✓  ✗  ✗]                                                                                                                                                                                  
  query 6 [✗  ✗  ✗  ✗  ✓  ✓  ✓  ✗]                                                                                                                                                                                  
  query 7 [✗  ✗  ✗  ✗  ✗  ✓  ✓  ✓]                                                                                                                                                                                  
                                                                                                                                                                                                                    
  可以看到 ✓ 形成了一条沿对角线的斜带（band），这就是名字的来源。                                                                                                                                                   
                                                                                                                                                                                                                    
  规则就是：query i → kv[i-window_size+1 .. i]                                                                                                                                                                      
                                                                                                                                                                                                                    
  对 DeepSeek-V4，window_size=128，即每个 query 向左看 128 个 token。                                                                                                                                               
                                                                                                                                                                                                                    
  ---             
  Band 模式为什么在 CP rank r>0 下出错                                                                                                                                                                              
                                              
  问题就出在：band 模式是基于 query 在 kv 中的绝对位置来算的，它假设 query i 对应 kv[i]。                                                                                                                           
                                              
  但 CP rank r>0 的 kv_full 布局是：                                                                                                                                                                                
                                          
  kv_full = [ boundary_128(来自rank r-1) | local_chunk(rank r自己) ]                                                                                                                                                
  位置索引:      0, 1, ..., 127          | 128, 129, ..., 128+chunk-1                                                                                                                                               
                                                                                                                                                                                                                    
  kv_full[128] 才是 rank r 的第 0 个 token（全局位置 r*chunk）。                                                                                                                                                    
                                                                                                                                                                                                                    
  此时 query 0（全局位置 r*chunk）需要的窗口是 kv_full[1..128]，但 band 模式算出的是 kv_full[i-127..i] = kv_full[-127..0]，负数索引无意义，完全错误：                                                               
                                                                                                                                                                                                                    
  正确窗口（需要）:  kv_full[1 .. 128]   ← 含边界 token                                                                                                                                                             
  band 模式给的:    kv_full[-127 .. 0]  ← 完全错位                                                                                                                                                                  
                                              
  这就是为什么 CP rank r>0 必须退回 SDPA，用显式传入的 window_topk_idxs 来指定正确的访问范围，而不能依赖 SFA kernel 的 band 模式自动计算。
